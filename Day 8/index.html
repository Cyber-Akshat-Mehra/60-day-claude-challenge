      🧪 Data Quality & Methodology

      How this dataset was built

      AQI values for all 10 cities were pulled from live public air-quality dashboards on 15 Jul 2026. PM2.5 and PM10 were not uniformly available at city level in a single query, so they were derived from each city's AQI using the CPCB sub-index breakpoint tables (reverse-interpolated), then cross-checked against a typical Indian urban PM2.5:PM10 ratio (~0.55). Water Quality Index uses CPCB / NITI Aayog composite state-level scores as a city-level proxy where a direct river/groundwater monitoring station wasn't cited — these are flagged estimated in the city cards. Missing/duplicate readings across sources were reconciled by preferring the most recent same-provider snapshot; no city was dropped, and no values were fabricated beyond the documented interpolation method.

      Note: because this is a static, single-snapshot dashboard (not a live API feed), the numbers won't refresh in your browser — re-run the analysis for a current read.

      Sources

      • AQI.in — India live dashboard
      • AQICN — India real-time map
      • CPCB — National Air Quality Index
      • CPCB — National Water Monitoring Programme data
